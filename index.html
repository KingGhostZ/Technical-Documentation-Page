Data

Python is known for it's easy use for storing and analyzing data. Whether it's is in
external modules and files or the lists and dictionaries. Python stores data typically in lists,
dictionaries, tuples, or sets. A list is unordered data that is stored within a python
variable. For example, you can make the variable a equal to integers, floating point
numbers, strings, other lists, and boolean data types. Which makes lists extremely easy to utilize.
They utilize the square brackets in order to store their information.
Dictionaries store data with key value pairs inside of curly brackets. For example,
you store a key value like city with the city you enter in. A tuple is a collection
of data that cannot be altered like coordinates. They also utilize
paranthesis to store their information. The last data collection type is a set which is
unordered, unchangeable, and unindexed. Sets utilize the curly brackets just like dictionaries.

Loops

Loops are a conditional statement where the computer will continue to run back
through the same block of code until it no longer is true. A while loop loops
through a block of code unlike a for loop that loops through a string or data type.
For example, while true, the while statement will continue to add numbers of
your choice until the statement is false. For loops will loop through a string or list that
will go through each letter or index to check or print that out. Here are some
examples down below...

  • x=o while x <= 10: print(x) x +=1
  • x="string" for x in x: print(x)

    • The first line of code will print out numbers 1 until it is larger than 10. The
    second block of code will print out each letter of that string. Saying s t r i n g.
    This could be very useful for large amounts of data within a list.

Classes

Classes can be complicated but one of the most powerful things in python. Python
is an object-oriented programming language and this is truly where the object comes
into Python. Classes are like the blueprints or the creation of the objects. They are
similar to creating your own data type within python. Classes utilize functions
in order to create certain functionalities of these objects. Such as a specific
student is an object. However, the student genre would be the class. Here is an
example below...

  • class student: def __init__(self, name, grade, age, gpa): self.name = name self.grade = grade self.age = age self.gpa = gpa
  • student1 = student("Jacob", 2, 8, 4.0) print(student1.name)

    • The first block of code is creating the class or the data type of a student.
    With the initialized function we can create the parameters of this object such as the
    name, grade, age, and gpa of an individual student. Then, we can finally create the
    object by assigning data to each of these parameters of this function inside of a
    variable. Which we do in the second block of code.
